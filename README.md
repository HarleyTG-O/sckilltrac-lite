## 📥 Installation

### System Requirements:
- **OS:** Windows 10/11 (64-bit)
- **RAM:** 4GB minimum, 8GB recommended
- **Internet:** Required (broadband recommended)
- **Star Citizen:** LIVE, PTU, or EPTU

### Installation Steps:
1. Download the latest release from [GitHub Releases](https://github.com/HarleyTG-O/sckilltrac-lite/releases)
2. Run `SCKillTrac-LITE-vX.X.X.exe`
3. Enter your license key when prompted
4. Accept the Terms of Service, Privacy Policy, and EULA
5. The app will auto-detect your Star Citizen installation
6. Start playing Star Citizen!

## 🎮 Usage

### First Launch:
1. **License Validation** - Enter your unique license key
2. **Welcome Screen** - Select language and preferences
3. **Log Detection** - App auto-detects Star Citizen logs
4. **Overlay Enabled** - Overlay appears when in-game

### Hotkeys:
- **Ctrl + Insert** - Toggle overlay lock/unlock (click-through mode)
- **Close Button (×)** - Hide overlay (relaunch app to show again)

### Overlay States:
- 🔒 **LOCKED** - Click-through mode (transparent to mouse)
- 🔓 **UNLOCKED** - Interactive mode (draggable, resizable)

## 🤝 Contributing to NPC Database

Your gameplay automatically contributes to the community NPC database:

### How It Works:
1. You encounter NPCs in Star Citizen
2. LITE detects actor names from game logs
3. AI classifier determines if it's an NPC
4. Unknown NPCs are reported to the database
5. Database is updated for all users

### Contribution Details:
- ✅ Automatic (no action required)
- ✅ Improves detection for everyone
- ✅ NPC names are public (GitHub)
- ❌ Cannot be retracted once submitted

**Database:** https://github.com/HarleyTG-O/sc-killfeed

## ⚖️ Ban System

SC Kill Tracker LITE enforces a comprehensive ban system:

### Ban Types:
| Type | Scope | Effect |
|------|-------|--------|
| **User Ban** | Username/SCKillTrac ID | Prevents login with banned username |
| **License Key Ban** | Specific license | License revoked, prevents activation |
| **HWID Ban** | Hardware ID | Blocks all licenses on device |
| **IP Ban** | IP Address | Temporary or permanent block |
| **Account Ban** | SCKillTrac ID | Most severe, cross-service ban |

### Ban Reasons:
- EULA violations
- Terms of Service violations
- License sharing or fraud
- Cheating or exploiting
